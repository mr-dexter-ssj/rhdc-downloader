# rhdc-downloader
A python script that allows you to use the Play now/Download buttons in romhacking.com without using Parallel Launcher
# Warning
This program is still under development and should not be used yet.
# Compiling
* Clone this repository
* Install PySide6, requests and pypatchergba
```pip install PySide6 requests pypatchergba```
* Run `python3 main.py --help` for a list of arguments and usage instructions
